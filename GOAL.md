The goal is to build a system which can generate agentic documentation for a given Red Hat OpenShift repository.
Refer the AGENT_KNOWLEDGE_FRAMEWORK.md file. Refer the skills guides in https://resources.anthropic.com/hubfs/The-Complete-Guide-to-Building-Skill-for-Claude.pdf and https://developers.openai.com/cookbook/examples/skills_in_api and synthesize the claude skills of each agent to be used by claude.
These skills should be used by claude to generate the agentic documentation for a given Red Hat OpenShift repository. 
The system should generate agentic documentation which should be used by coding agents to create new features and solve bug fixes.
Knowledge graphs should be generated by using the graphify (https://github.com/safishamsi/graphify) tool exposed as a claude skill to be loaded in claude.
The generated agentic documentation should follow the guidelines and structure outlined in https://github.com/Prashanth684/agentic-docs-guide.
The directory structure of the agentic documentation generated should follow the same structure as outlined in https://github.com/Prashanth684/agentic-docs-guide/blob/main/AGENTIC_DOCS_FRAMEWORK.md
The github and JIRA data should be ingested efficiently using the github and JIRA MCP servers.
There should be accurate and reliable monitoring and logging.
The agentic documentation should be retrieved by coding agents only through the retrieval agent.
Each task and operation performed by the retrieval agent should be logged and monitored.
The validator agent should contain well defined skills to evaluate the quality of the agentic documentation generated.
The validator agent should contain well defined skills to evaluate the quality of the knowledge graph generated.
The validator agent should contain a feedback loop to the orchestrator agent to trigger the documentation generation again until the documentation is generated to the desired quality.
The system should follow the skills structure, project structure, directory structure and guidelines as outlined in https://github.com/addyosmani/agent-skills repository.
